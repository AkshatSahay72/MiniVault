Mini-Vault

Mini-Vault is a simple text-based note storage and search system.

It allows you to:
- Add text notes
- Store notes in a SQLite database
- Search notes using TF-IDF and cosine similarity

Tech Stack:
- Python
- SQLite
- Flask or FastAPI
- scikit-learn

Project Structure:
mini-vault/
- app.py        : API entry point
- db.py         : Database logic
- vectorizer.py : TF-IDF vectorization
- search.py     : Search logic
- README.txt

Current Status:
- Database implemented
- Note insertion working
- Fetch all notes working

Purpose:
Build a small, complete backend project and practice core concepts.

Usage:
Run the API, add notes, search notes.

End.
